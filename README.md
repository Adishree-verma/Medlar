💜 Medlar in Love

«Understand conversations. Build healthier connections. Communicate with confidence.»

Medlar in Love is an AI-powered relationship companion built with modern web technologies. It analyzes conversations in real time to identify communication patterns, relationship signals, and provide personalized insights to help users develop healthier and more confident interactions.

The platform combines artificial intelligence with conversational analysis to help users better understand their communication style, recognize positive and negative patterns, and improve the way they connect with others.

---

✨ Features

- 🤖 AI-Powered Conversation Analysis
  
  - Analyze conversations and extract meaningful insights.

- 🟢🟡🔴 Green, Yellow & Red Flag Detection
  
  - Identify potential relationship signals based on conversation patterns.

- 💬 AI Reply Suggestions
  
  - Get personalized suggestions to improve communication.

- ✨ Rizz Evaluation
  
  - Analyze conversation confidence and messaging style.

- ❤️ Relationship Stage Detection
  
  - Understand interaction patterns and relationship progression.

- 📊 Communication Analytics
  
  - Track conversation trends and behavioral patterns.

- 🎯 Personalized Recommendations
  
  - Receive AI-generated insights based on conversations.

🚀 Getting Started

Prerequisites

- Node.js (LTS recommended)
- npm / yarn / pnpm

Installation

git clone <repository-url>

cd medlar

npm install

Development Server

npm run dev

Open your browser and visit:

http://localhost:3000
